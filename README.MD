# PPH21 Calculator (PHP)

Project sederhana untuk menghitung PPH21 menggunakan PHP OOP. Project ini hanya untuk belajar penerapan OOP dalam PHP, referensi dari SantriKoding.com

## Fitur
- Perhitungan PPH21 berdasarkan PKP
- Menggunakan OOP, Interface, Abstract Class
- Namespace PHP

## Cara Menjalankan
1. Clone repository
2. Jalankan:
   php index.php
